# Simhub-ETS-ATS-Actros-Dashboard


Mercedes Benz Actros Dashboard

Its has multiple pages left and right from the speedometer

Left Side:
- Fuel
- Wear
- Temps and Air Pressure

Right Side:
- Job Distance
- Job Info
- JIT / Sleep
- Fines, Tolls, Tickets
